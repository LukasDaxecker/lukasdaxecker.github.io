Einleitung

Willkommen bei den Techniktagen 2025! In diesem Kurs lernen wir die Grundlagen der Programmierung mit Python. Python ist eine der beliebtesten Programmiersprachen der Welt, da sie sehr mächtig und gleichzeitig leicht zu lesen ist.

Grundlagen

Variablen & Datentypen

Variablen sind wie Boxen oder Container, in denen wir Daten speichern können, um sie später wiederzuverwenden. Jede Variable hat einen Namen und einen Wert.

Wichtige Datentypen:

  • int (Integer): Ganze Zahlen (z. B. 5, -10, 42)
  • float: Kommazahlen (z. B. 3.14, -0.01) – Achtung: Punkt statt Komma!
  • str (String): Textzeichenketten (z. B. "Hallo", 'Python')
  • bool (Boolean): Wahrheitswerte (True, False)

Code-Beispiel:

alter = 20 # Das ist ein Integer name = "Lukas" # Das ist ein String groesse = 1.85 # Das ist ein Float

Regeln für Variablennamen:

  • Erlaubt sind Buchstaben (a-z, A-Z), Zahlen (0-9) und der Unterstrich (_).
  • Ein Name darf nicht mit einer Zahl beginnen.
  • Groß- und Kleinschreibung ist wichtig: alter und Alter sind zwei verschiedene Variablen!
  • Reservierte Wörter (wie if, else, for, print) dürfen nicht als Name verwendet werden.

Ein- und Ausgabe

Damit unser Programm mit dem Benutzer interagieren kann, nutzen wir zwei grundlegende Befehle:

  • print(): Gibt Text oder Werte auf dem Bildschirm aus.
  • input(): Wartet auf eine Eingabe des Benutzers über die Tastatur.

Wichtig: Alles, was wir mit input() einlesen, ist zunächst immer Text (String). Wenn wir damit rechnen wollen, müssen wir es umwandeln (z. B. mit int()).

# Ausgabe print("Hello World") # Eingabe name = input("Bitte gib deinen Namen ein: ") # Ausgabe mit Variablen (f-String) print(f"Dein Name ist {name}")

Kommentare

Kommentare sind Notizen im Code, die vom Computer ignoriert werden. Sie helfen uns Menschen zu verstehen, was der Code tut.

# Das ist ein einzeiliger Kommentar. Alles dahinter wird ignoriert. """ Dies ist ein mehrzeiliger Kommentar. Er kann über mehrere Zeilen gehen. Sehr praktisch für längere Erklärungen. """

Übung 1: Alter und Name

Erstelle ein Programm, das den Benutzer nach seinem Namen und seinem Alter fragt und ihn dann freundlich begrüßt.

Beispielablauf:
Eingabe: Gib deinen Namen ein: Lukas
Eingabe: Gib dein Alter ein: 20
Ausgabe: Hallo Lukas, du bist 20 Jahre alt.

Lösung anzeigen

name = input("Gib deinen Namen ein: ") alter = input("Gib dein Alter ein: ") print(f"Hallo {name}, du bist {alter} Jahre alt.")

Rechenoperatoren

Computer sind im Grunde extrem schnelle Taschenrechner. In Python können wir ganz einfach rechnen.

Die wichtigsten Operatoren:

  • + : Plus (Addition)
  • - : Minus (Subtraktion)
  • * : Mal (Multiplikation)
  • / : Geteilt (Division) - Ergebnis ist immer eine Kommazahl
  • ** : Potenzieren (z. B. 3 ** 2 ist 3^2 also 9)

Code-Beispiel:

x = 10 y = 5 summe = x + y # Ergebnis: 15 produkt = x * y # Ergebnis: 50 potenz = x ** 2 # Ergebnis: 100 (10 hoch 2) print(summe)

Hinweis: Wie in der Mathematik gilt auch hier "Punkt vor Strich". Man kann Klammern () nutzen, um die Reihenfolge zu ändern.

Übung 2: Geometrie-Rechner

Jetzt wenden wir das Gelernte an. Schreibe ein kleines Programm, das Flächen berechnet. Der Benutzer soll die Werte eingeben.

Teil A: Das Rechteck

Frage nach Länge und Breite und berechne:

  • Fläche: A = L * B
  • Umfang: U = 2 * (L + B)
Lösung Rechteck

# Wir brauchen float(), da Maße auch Kommazahlen sein können laenge = float(input("Länge eingeben: ")) breite = float(input("Breite eingeben: ")) flaeche = laenge * breite umfang = 2 * (laenge + breite) print(f"Fläche: {flaeche}") print(f"Umfang: {umfang}")

Teil B: Der Kreis

Frage nach dem Radius und berechne die Fläche.

Formel: A = π * r2

Tipp: Nutze für π einfach den Wert 3.14.

Lösung Kreis

radius = float(input("Radius eingeben: ")) # Fläche = Pi * r hoch 2 flaeche = 3.14 * (radius ** 2) print(f"Die Kreisfläche beträgt: {flaeche}")

Teil C: Das Dreieck

Frage nach Grundlinie (g) und Höhe (h).
Formel: (g * h) / 2

Lösung Dreieck

grundlinie = float(input("Grundlinie: ")) hoehe = float(input("Höhe: ")) flaeche = (grundlinie * hoehe) / 2 print(f"Dreiecksfläche: {flaeche}")

Verzweigungen (If-Else)

Mit if (wenn) und else (sonst) können wir Entscheidungen treffen. Der Computer prüft eine Bedingung. Ist sie wahr, wird der erste Block ausgeführt, sonst der zweite.

Vergleichsoperatoren:

  • == (gleich)
  • != (ungleich)
  • < (kleiner als), <= (kleiner gleich)
  • > (größer als), >= (größer gleich)

Beispiel:

name = input("Wie heißt du? ") if name == "Lukas": print("Hey, du heißt ja wie ich!") else: print("Schöner Name!")

Achte auf die Einrückung (Tabulator) nach dem Doppelpunkt! In Python bestimmt die Einrückung, was zum if-Block gehört.

Übung 3: Der Türsteher

Schreibe ein Programm, das prüft, ob eine Person volljährig ist.

Beispielablauf:
Eingabe: Gib dein Alter ein: 17
Ausgabe: Du bist noch nicht erwachsen.

Lösung anzeigen

# Wir müssen die Eingabe in eine Zahl umwandeln: int() alter = int(input("Gib dein Alter ein: ")) if alter >= 18: print("Du bist Erwachsen") else: print("Du bist nicht Erwachsen")

Erweiterte Bedingungen (elif, and, or)

Manchmal gibt es mehr als nur zwei Möglichkeiten ("Ja" oder "Nein"). Dafür nutzen wir elif (else if).

Außerdem können wir Bedingungen verknüpfen:

  • and: Beide Bedingungen müssen wahr sein.
  • or: Mindestens eine Bedingung muss wahr sein.

Beispiel mit and (Bereichsprüfung):

alter = int(input("Gib dein Alter ein: ")) if alter >= 13 and alter <= 19: print("Du bist ein Teenager.") else: print("Du bist kein Teenager.")

Beispiel mit elif (Mehrere Stufen):

health = 75 if health == 100: print("Perfekt! Volle Gesundheit.") elif health > 50: print("Dir geht es gut.") else: print("Achtung! Wenig Leben übrig.")

Zufallszahlen

Um Spiele spannender zu machen, brauchen wir den Zufall. In Python importieren wir dafür das Modul random.

import random # Erzeugt eine zufällige ganze Zahl zwischen 1 und 100 zahl = random.randint(1, 100) print(f"Die Zufallszahl ist: {zahl}")

Übung 4: Zahlenraten I

Der Computer denkt sich eine Zahl zwischen 1 und 10 aus. Der Spieler muss sie erraten.

Anforderung:
Das Programm soll sagen, ob man richtig geraten hat oder nicht. Falls nicht, soll die richtige Zahl verraten werden.

Lösung anzeigen

import random geheimzahl = random.randint(1, 10) tipp = int(input("Rate eine Zahl zwischen 1 und 10: ")) if tipp == geheimzahl: print("Super! Du hast die Zahl erraten.") else: print(f"Leider falsch. Die Zahl war {geheimzahl}.")

Schleifen (Loops)

Schleifen verwenden wir, wenn Code mehrfach ausgeführt werden soll, ohne dass wir ihn oft kopieren müssen. Die while-Schleife läuft so lange, wie ihre Bedingung wahr ist.

Beispiel:

zaehler = 0 # Solange der Zähler kleiner als 3 ist... while zaehler < 3: print(f"Durchlauf Nummer {zaehler}") zaehler = zaehler + 1 # Wichtig: Zähler erhöhen, sonst läuft sie ewig!

Übung 5: Die Wiederholung

Lass den Benutzer ein Wort und eine Anzahl eingeben. Das Programm soll das Wort so oft wiederholen, wie gewünscht.

Eingabe: Wort: "Hallo", Anzahl: 3
Ausgabe:
Hallo
Hallo
Hallo

Lösung anzeigen

wort = input("Welches Wort soll wiederholt werden? ") anzahl = int(input("Wie oft? ")) i = 0 while i < anzahl: print(wort) i = i + 1

Übung 6: Zahlenraten II

Erweitere das Spiel aus Übung 4. Der Spieler hat nun 3 Versuche, um die Zahl zu erraten.

Tipp: Verwende eine Schleife, die maximal 3 Mal läuft, oder abbricht, wenn die Zahl erraten wurde (break).

Lösung anzeigen

import random geheimzahl = random.randint(1, 10) zaehler = 0 while zaehler < 3: tipp = int(input("Rate eine Zahl zwischen 1 und 10: ")) if tipp == geheimzahl: print("Super! Du hast die Zahl erraten.") break; else: print(f"Leider falsch. Die Zahl war {geheimzahl}.") zaehler = zaehler + 1

Übung 7: Zahlenraten III

Erweitere das Spiel weiter: Wenn der Spieler falsch rät, gib ihm einen Hinweis, ob die gesuchte Zahl größer oder kleiner ist als sein Tipp.

Lösung anzeigen

import random geheimzahl = random.randint(1, 10) zaehler = 0 while zaehler < 3: tipp = int(input("Rate eine Zahl zwischen 1 und 10: ")) if tipp == geheimzahl: print("Super! Du hast die Zahl erraten.") break; elif tipp < geheimzahl: print(f"Leider falsch. Die Geheimzahl ist groesser.") zaehler = zaehler + 1 elif tipp > geheimzahl: print(f"Leider falsch. Die Geheimzahl ist kleiner.") zaehler = zaehler + 1

Übung 8: Taschenrechner I

Das Ziel ist es einen Taschenrechner für Formen zu bauen.

Anforderung:
Der Nutzer soll zuerst eingeben können, von welcher Form (Dreieck, Kreis, Quadrat, ...) die Fläche berechnet werden soll und dannach die maße für besagte Figur eingeben.

Tipp: Um es leichter zu machen, weise jeder Form eine Zahl zu.

Lösung anzeigen

import math # Damit math.pi verwendet werden kann, ansonsten statt pi 3.14 verwenden form = int(input("Gib eine Form ein (1 = Dreieck, 2 = Rechteck, 3 = Kreis): ")) if form == 1: ha = int(input("Gib die Höhe ein: ")) a = int(input("Gib die Länge ein: ")) area = a * ha / 2 print(area) elif form == 2: b = int(input("Gib die Breite ein: ")) a = int(input("Gib die Länge ein: ")) area = a * b print(area) elif form == 3: r = int(input("Gib den Radius ein: ")) area = math.pi * r * r print(area) else: print("Form exestiert nicht!")
