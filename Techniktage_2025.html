Willkommen zur Coding Academy.

Python

Variablen

Datentypen: int, bool, float, string, char, ...

alter = 20 name = "Lukas"

Regeln: Symbole a-z, A-Z, 0-9 und _. Es dürfen keine Zahlen am Beginn des Namens stehen und manche Wörter (if, for, while) können nicht verwendet werden.

Außerdem sind zum Beispiel alter und Alter unterschiedliche Variablen.

Aus- Eingabe

print, input

print("Hello World") name = input("Bitte gib deinen Namen ein: ") # Type conversion print(name) print(f"Dein Name ist {name}")

Kommentare

Aufgabe 1 - Alter und Name

Eingabe:
Gib deinen Namen ein:
Gib dein Alter ein:

Ausgabe:
Hallo, Lukas du bist 20

Lösung:

name = input("Gib deinen Namen ein: ") alter = input("Gib dein Alter ein: ") print(f"Hallo {name}, du bist {alter} Jahre alt")

If-Else Bedingung

Zwei oder mehrere Dinge unterscheiden.

Operatoren: ==, !=, <=, >=, <, >

Beispiel:

if 10 == 10: print("Hi") name = input("Gib deinen Namen ein: ") if name == "Lukas": print("Du heißt Lukas") else: print("Du heißt nicht Lukas")

Aufgabe 2 - Erwachsen

alter = int(input("Gib dein Alter ein: ")) if alter >= 18: print("Du bist Erwachsen") else: print("Du bist nicht Erwachsen")

elif, and, or

alter = int(input("Gib dein Alter ein: ")) if alter >= 13 and alter <= 19: print("Du bist ein Teenager") else: print("Du bist kein Teenager")

Aufgabe 3 - Zahlen raten I

import random zahl = random.randint(1,10) geraten = int(input("Gib eine Zahl ein: ")) if zahl == geraten: print("Du hast die Zahl erraten") else: print(f"Du hast die Zahl nicht erraten. Die Zahl war {zahl}")

Schleifen

zaehler = 0 while zaehler < 3: print("Hallo") zaehler = zaehler + 1

Aufgabe 4 - Schleifen

wiederholungen = int(input("Gib die Anzahl der Wiederholungen an: ")) wort = input("Gib ein Wort ein: ") zaehler = 0 while zaehler < wiederholungen: print(wort) zaehler = zaehler + 1
